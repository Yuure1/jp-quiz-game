# Japanese Quiz Game

Want to put your knowledge in hiragana and katakana to the test? Check this game out!

First, pick which writing system you'd like to have a quiz on. Press the 'あ' button for hiragana and the 'カ' button for katakana.

![Select writing system](/imgs/Pick%20a%20writing%20system.png)

Then choose how many items you'd like. 

![Choose quiz length](/imgs/Choose%20quiz%20length.png)

The options give the following amount of items:
- **SHORT** = 10 items
- **HALF** = 23 items
- **FULL** = 46 items

> Note: As of now, the game does not include characters with dakuten( ﾞ) and handakuten( ﾟ) as well as combination characters but they will be added in the future.
